Latest Projects

Key Logger Project

This project is a simple JavaScript-based key logger designed for educational purposes. It allows users to capture and display keypress events in real-time. Users can start and stop the logging process using buttons, with clear visual feedback provided for each key press and release. Features: Key Logging: Captures keydown and keyup events, displaying which key is pressed or released. User Interface: A clean and responsive layout with buttons to control the logging. Feedback Messages: Provides instructional messages and updates on the logging state. This project serves as a great introduction to handling keyboard events in JavaScript and demonstrates basic event listeners and DOM manipulation techniques.

Typing Speed Tester

Purpose: A web-based typing speed test to assess and improve typing skills. Functionality: Users type displayed text for a set duration (60 seconds). Tracks and displays typing speed (WPM), errors, and time left. User Interface: Clean design with a text input field and real-time feedback. Results displayed dynamically as users type. Features: Randomized text for varied practice. "Try Again" button to restart the test. This project is a fun way to practice typing while learning about event handling and DOM manipulation in JavaScript!

Expense Tracker

Purpose: A web-based application to help users track their expenses. Functionality: Users can add expenses with descriptions, categories, and amounts. Displays a summary table of all recorded expenses. Calculates and shows the total expense amount dynamically. User Interface: Simple and clean design with a form for input and a table for displaying expenses. Features: Input validation to ensure all fields are filled correctly. Responsive layout for ease of use. This project is a practical tool for personal finance management and demonstrates basic HTML, CSS, and JavaScript skills!
